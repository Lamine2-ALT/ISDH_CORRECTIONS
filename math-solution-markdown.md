## **1°) SIMPLIFICATION DE t**

$$t=\sqrt{45}+\sqrt{196}-\sqrt{180}-\sqrt{245}$$

$$t = \sqrt{9 \times 5}+\sqrt{196}-\sqrt{36 \times 5}-\sqrt{49 \times 5}$$

$$t = 3\sqrt{5} + 14 - 6\sqrt{5} - 7\sqrt{5}$$

$$t = 3\sqrt{5} - 6\sqrt{5} - 7\sqrt{5} + 14$$

$$t = (3 - 6 - 7)\sqrt{5} + 14$$

$$t = -10\sqrt{5} + 14$$

**Donc $a = -10$, $b = 5$ et $c = 14$**

## **2°) ÉTUDE DE x ET y**

### **a°) Écrire x avec un dénominateur rationnel**

$$x=\frac{4}{7+3\sqrt{5}}$$

$$x = \frac{4}{7+3\sqrt{5}} \times \frac{7-3\sqrt{5}}{7-3\sqrt{5}}$$

$$x = \frac{4(7-3\sqrt{5})}{(7+3\sqrt{5})(7-3\sqrt{5})}$$

On utilise l'identité remarquable $(a+b)(a-b) = a^2 - b^2$ avec $a = 7$ et $b = 3\sqrt{5}$ :

$$x = \frac{4(7-3\sqrt{5})}{7^2 - (3\sqrt{5})^2}$$

$$x = \frac{4(7-3\sqrt{5})}{49 - 9 \times 5}$$

$$x = \frac{4(7-3\sqrt{5})}{49 - 45}$$

$$x = \frac{4(7-3\sqrt{5})}{4}$$

$$x = 7-3\sqrt{5}$$

---

### **b°) Montrer que y est négatif**

On a : $y=3\sqrt{5}-7$

Pour comparer $3\sqrt{5}$ et $7$, je compare leurs carrés car ils sont positifs :

$$(3\sqrt{5})^2 = 9 \times 5 = 45$$

$$7^2 = 49$$

On a : $45 < 49$ 

Donc $(3\sqrt{5})^2 < 7^2$

Comme deux nombres positifs sont rangés dans le même ordre que leurs carrés : $3\sqrt{5} < 7$

Donc : $3\sqrt{5} - 7 < 0$

**Conclusion : y est négatif**

---

### **c°) Montrer que x = -y**

$$-y = -(3\sqrt{5}-7)$$

$$-y = -3\sqrt{5}+7$$

$$-y = 7-3\sqrt{5}$$

Or d'après a) : $x = 7-3\sqrt{5}$

**Donc $x = -y$**

### **d°) Encadrer x à $10^{-2}$ près**

On a : $2,236< \sqrt{5}< 2,237$

$$3 \times 2,236 < 3\sqrt{5} < 3 \times 2,237$$

$$6,708 < 3\sqrt{5} < 6,711$$

$$-6,711 < -3\sqrt{5} < -6,708$$

$$7 - 6,711 < 7-3\sqrt{5} < 7 - 6,708$$

$$0,289 < x < 0,292$$

**A $10^{-2}$ près : $0,28 < x < 0,30$**

## **3°) MONTRER QUE $\sqrt{z}=-2y$**

On a : $z=(x-y)^{2}$

D'après 2°c), on sait que $x = -y$

Donc : 

$$z = (-y - y)^{2}$$

$$z = (-2y)^{2}$$

$$z = 4y^{2}$$

Alors : 

$$\sqrt{z} = \sqrt{4y^{2}}$$

$$\sqrt{z} = 2\sqrt{y^{2}}$$

$$\sqrt{z} = 2|y|$$

D'après 2°b), $y < 0$ donc $|y| = -y$

$$\sqrt{z} = 2 \times (-y)$$

$$\sqrt{z} = -2y$$

**On a bien $\sqrt{z}=-2y$**
